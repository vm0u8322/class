# MochiClass

MochiClass 是一個以 **VaultSage API** 為核心的學生課堂檔案整理工具。它打破了傳統筆記軟體先上傳再分類的思維，改以「課表」作為資料管理的入口，幫助學生自動化整理雜亂的課堂影音與文件。

## 線上演示

您可以直接造訪此網址體驗：[MochiClass 線上版本](https://vm0u832-mochiclass.hf.space/)

## API 用途與核心功能

本專案主要結合 VaultSage API 來達成智慧化的助理功能：

- **課表自動解析**：透過 VaultSage API 的自然語言處理能力，將非結構化的文字課表（甚至是亂序的）解析成結構化的課程時段資料。
- **雲端資料夾管理**：自動在 VaultSage 雲端建立對應課程的資料夾，實現「課程即資料夾」的隔離管理。
- **文件分析與問答**：利用 API 的檢索增強生成（RAG）功能，針對特定課程已上傳的講義、OCR 照片內容與錄音逐字稿，提供精準的問答、重點整理與追問功能。

## 創意與特色

### 1. 以「課表」為核心的檔案入口
學生上課時產生的資料非常零散：黑板照片、老師隨手發的 PDF、長達一小時的錄音。MochiClass 讓學生「點選課表上的某一節課」，就能直接看到所有跟該課相關的資料，減少手動建立資料夾與命名的負擔。

### 2. 多維度自動配對技術
系統不依賴檔名分類，而是結合以下資訊找出檔案屬於哪堂課：
- **時間維度**：比對檔案建立時間與課表時段。
- **內容維度**：透過 OCR 圖片識別與語音轉文字（Whisper）抓取關鍵字，與課程名稱進行二次比對。
- **問答維度**：利用 VaultSage 的廣泛上下文理解，進行精準的資料關聯。

### 3. 本機處理與雲端智慧的協作
- **本機端**：負責圖片 OCR、語音轉文字、檔案壓縮與本地狀態暫存，確保操作流暢。
- **雲端（VaultSage）**：負責長期的結構化儲存與深度語義問答，提供強大的大模型整理能力。

## 詳細操作說明

1. **設定 API Key**：啟動後透過設定介面（或建立 `.env` 檔案）輸入您的 VaultSage API Key。
2. **匯入課表**：在左側課表區域貼上文字課表，按下「AI 解析課表」。
3. **上傳課堂檔案**：將範例講義、錄音或課堂照片（jpg/png/pdf/wav）整批上傳。
4. **系統自動分類**：系統會根據時間與內容，自動將檔案掛載到對應的課程格子中。
5. **瀏覽與預覽**：點選課表上的格子，右側會顯示該課的所有檔案。點擊照片可直接展開預覽。
6. **智慧總結與問答**：點選下方的「AI 整理與問答」，輸入問題（如「這堂課的重點是什麼？」），系統會根據該課程的資料給出答案。
7. **一鍵導出**：若需要將該課程資料打包帶走，可使用下載 ZIP 功能獲取該課所有圖片與錄音。

## 本機執行環境要求

```powershell
# 1. 安裝虛擬環境與套件
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt

# 2. 啟動伺服器
python server.py
# 或使用 uvicorn
uvicorn server:app --host 127.0.0.1 --port 4180
```

啟動後請開啟瀏覽器至 `http://127.0.0.1:4180` 即可開始使用。
